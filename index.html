Leaf

Super simple web application framework built on Guice

Download this project as a .zip file Download this project as a tar.gz file

Leaf is an extremely simple Java web framework which takes concepts from Google Sitebricks and Apache Wicket, rolls them together, chucks out the complicated bloat, mixes in a few new ideas and makes sure the whole lot runs really well on Google App Engine or any servlet container.

It is built on top of Google's Guice Servlet framework so you get all of that dependency injection goodness and easy configuration.

Leaf is simple - a lot more simple than most web frameworks because it does not try to replicate features that can be best handled with GWT on the client.

Overview

Like Apache Wicket, Leaf uses its own super simple markup template system that matches an id in your well formed markup to a Java component that can modify that markup. There is no scripting language. Scripting in markup files is bad. The attached Java component controls all modifications to the markup and you can debug all of it in your IDE.
